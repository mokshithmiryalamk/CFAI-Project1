\# Algorithm Comparison Dashboard



A Flask-based web application that compares frequency counting algorithms and displays execution time using a dashboard.



\## Features

\- Nested loop frequency counting

\- HashMap frequency counting

\- Execution time comparison

\- Chart visualization

\- Random input generation



\## Technologies Used

\- Python Flask

\- HTML

\- CSS

\- JavaScript

\- Chart.js



\## How to Run

```bash

python app.py
